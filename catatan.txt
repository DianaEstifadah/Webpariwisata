Catatan:

1. Buat database dengan nama 'wisataweb', lalu import file wisataweb.sql
2. Pada halaman home berisi informasi tentang berbagai tempat wisata.
3. Pengguna dapat membuat artikel dengan mengisi form pembuatan artikel
   yang ada pada halaman 'buat artikel'
4. Setelah membuat artikel dan mengirimnya, data tidak akan ditampilkan secara langsung,
   data akan menunggu persetujuan admin terlebih dahulu.
5. Untuk Admin bisa mengakses url http://localhost/Webpariwisata/su agar dapat mengakses halaman Admin.
6. - Admin dapat Login menggunakan username = 'Diana' & password = '123'
   - Admin dapat Login menggunakan username = 'Ita' & password = '123'
